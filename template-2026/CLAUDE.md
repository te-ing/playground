# Template 2026

React 19 + TypeScript + Vite 7 프로젝트 템플릿

## Quick Reference

```bash
npm run dev      # 개발 서버
npm run build    # 프로덕션 빌드
npm run lint     # ESLint 실행
npm run format   # Prettier 포맷팅
```

## Tech Stack

- React 19, TypeScript 5.9, Vite 7
- React Router 7, SCSS Modules
- ESLint + Prettier
